Project for the article entitled 
"Global loss of phylogenetic uniqueness by naturalized species across taxonomic groups"
by
Qi Yao, Lirong Cai, Shao-peng Li, and Patrick Weigelt

Authorship of code: R Code was written by Qi Yao

Details of R script and their function stored in the code file:

(1) 0. server/code/1. calculating_turnover_***.R - Calculating phylogenetic simpson, sorenson, jaccard, and ruggiero dissimilarity index for native and entire (natives + naturlized exotics) assemblages among regions, using phylogenetic trees and species inventory data worldwide for four main taxonomic groups (plants, birds, mammals, and freshwater fishes).

(2) 0. server/code/1. partitioning_***.R - Partitioning different sets of exotic species and calculating specific changes in PU to compare the relative magnitude of different pathways in driving changes in regional PU, also for four main taxonomic groups.

(3) 0. server/code/2. calculating_turnover_null_***.R - Recalculating phylogenetic dissimilarity for entire assemblages using modified naturalized species inventory data based on a series null models.

(4) 1. plotting_phy_beta_***.R - Plotting the patterns of PU for native, entire assemlages and changes (entire/native) for four main groups and four dissimilarity indices.
