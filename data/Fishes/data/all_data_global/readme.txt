This database is focused of freshwater fish species but can include also species using the marine environment to some extent

How to cite
To give due credit to the original authors, please cite data taken from the global fish distribution database as:
Tedesco PA, Beauchard O, Bigorne R, Blanchet S, Buisson L, Conti L, Cornu J-F, Dias MS, Grenouillet G, Hugueny B, Jézéquel C, Leprieur F, Brosse S & Oberdorff T. 2017. A global database on freshwater fish species occurrence in drainage basins. Scientific Data, 4: 170141. (v.1.2; 09/2019)

Disclaimer
The global database on freshwater fish species occurrence in drainage basins present information on fishes as correctly as possible. However, we can not exclude errors, and we can not be held responsible for any damage that may arise from these. For any improvement to the database that you could contribute with, please contact pablo.tedesco@ird.fr.

Legend of the columns

1.Basin.Name			Name of the drainage basin
2.Species.Name.in.Source	Name of the species as found in the data source
3.Native.Exotic.Status		Status of the species in the drainage basin (native, exotic)
4.TSN.ITIS.Code			Integrated Taxonomic Information System code of the species
5.Fishbase.Species.Code		Fishbase code of the species
6.Fishbase.Valid.Species.Name	Fishbase valid name of the species
7.Occurrence.Status		Status of the occurrence in the drainage basin (valid, questionable)
Freshwater			Fishbase habitat information for the species; presence in freshwaters
Brackish			Fishbase habitat information for the species; presence in brackish waters
Saltwater			Fishbase habitat information for the species; presence in marine waters
Family				Family name
Genus				Genus name
Fishbase_link			Link to the species in Fishbase
Morphology			Availability of morphological data in internal database (1 Yes, 0 No)
